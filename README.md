# Coffee Can Scrollytelling

A cinematic Next.js scrollytelling experience featuring a 192-frame canvas animation.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
```

This will create a static export in the `out/` directory.

## Features

- 192-frame canvas animation
- Scroll-driven frame playback
- Camera push-in effect
- Cinematic text overlays
- Dark luxury design
- Static export ready
